# Cookbook API

API приложения кулинарной книги на FastAPI.

## Возможности

- просмотр списка рецептов;
- просмотр детальной информации о рецепте;
- создание рецептов;
- подсчёт количества просмотров рецептов;
- сортировка рецептов по популярности.

## Стек технологий

- FastAPI
- SQLAlchemy Async
- SQLite
- Pydantic

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
uvicorn app.main:app --reload
```

## Документация

После запуска доступна по адресу:
